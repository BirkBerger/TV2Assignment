## Getting Started

# Weather widget

A simple React component that displays current weather conditions based on location input.
This repo serves as a solution to a coding test for [TV 2](https://tv2danmark.dk/karriere/ledige-stillinger/?hr=show-job/333138&locale=da_DK&linkref=683616&source=LinkedIn).

Built by **Thea Birk Berger**.

## Tech Stack
- Vite 6, React, TypeScript

Requires Node 22+

To get project

```bash
git clone https://github.com/BirkBerger/TV2Assignment
cd TV2Assignment
```

Create file .env.local in root
```bash
VITE_OPENWEATHER_API_KEY=<your-key>
```

To run project

```bash
npm install
npm run dev
```

To run unit test
```bash
npm test
```

To lint
```bash
npm run lint
```

Open http://localhost:5173 in your browser to see the result.
